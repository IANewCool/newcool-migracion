# NewCool Migracion - Visas y Residencia

Portal educativo sobre migracion y extranjeria en Chile.

## Demo

[https://migracion.newcool.io](https://migracion.newcool.io)

## Caracteristicas

- Buscador de oficinas de migraciones
- Guia de tipos de visa
- Calculadora de costos migratorios
- Proceso de residencia definitiva
- Permisos de trabajo

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

## Parte del Ecosistema NewCool

Este modulo es parte de [NewCooltura Informada](https://informada.newcool.io) - Portal de instituciones publicas de Chile.
